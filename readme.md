# SpeedReader

Drop in text and read ultra fast using RSVP (Rapid Serial Visual Presentation) technology.

## Features
- Paste any text, article, or document
- Adjustable reading speed (WPM)
- Clean, distraction-free interface
- Read 2-3x faster than traditional reading
